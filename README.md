# Excalidraw Drawing App

An Excalidraw-based drawing application with cloud storage using Vercel Blob.

## Features

- Full Excalidraw drawing interface
- Cloud storage with Vercel Blob
- Save, load, and manage drawings
- Light/Dark theme support
- Responsive design

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Configure BLOB_READ_WRITE_TOKEN in environment variables

## Deployment

This app is configured for automatic deployment on Vercel when changes are pushed to the main branch.
